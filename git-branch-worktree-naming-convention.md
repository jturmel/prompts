## Branch and Worktree naming convention

**Format:** `prefix/short-description` (lowercase, kebab-case)

| Prefix | Usage |
| :--- | :--- |
| `feature/` | New features |
| `fix/` | Bug fixes |
| `refactor/` | Code cleanup |
| `chore/` | Maintenance/Deps |
| `docs/` | Documentation |
| `hotfix/` | Urgent prod fix |
| `test/` | Test suites |

**Examples:**
- `feature/add-social-auth`
- `fix/save-blog-post-error`
- `refactor/validation-framework`
- `hotfix/login-crash`

**Worktree location:** `.worktrees/` in the root of the repo
